# simple_wp_boilerplate

Simple blank wordpress theme boilerplate
With Carbon fields
and Bootstrap

Version 1.2 includes Carbon Fields(v3) and example.php with simple starter for custom fields

**Install**
Firstly you need to install Carbon Fields:
composer install
